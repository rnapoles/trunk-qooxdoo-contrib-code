Javascript Development Toolkit (JSDT)

Removing a library from the Global Scope

  To remove a JavaScript Library from a project follow these instructions:

1. Right-click on the project and select Properties
 

2. Select the JavaScript Include Path properties page in the left navigation bar. Highlight the library you wish to remove and click Remove
 

3. Finally, click OK to close the JavaScript Include Path page.
