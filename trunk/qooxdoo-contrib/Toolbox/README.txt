===== Toolbox =====

This is an experimental project to allow for a browser-controlled usage of the
qooxdoo tool chain.

It is initially based upon two previous projects, which where moved from the 
qooxdoo framework repository into this contribution:

* buildtool (for qooxdoo 0.7.x) ==> Toolbox/0.1 
* toolbox (for qooxdoo 0.8.x)   ==> Toolbox/trunk
