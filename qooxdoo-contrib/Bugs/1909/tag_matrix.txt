== Problematic changes ==
Whenever the scaling is changed we face the problem. Although the scaling is (yet) not applied directly. It does only change if the developer sets another source - then the current scaling setting is used.

  * no-repeat -> repeat with setting of another source problematic in ALL browsers except when an alpha PNG is used in IE
  * PNG -> JPG in IE with scaling
