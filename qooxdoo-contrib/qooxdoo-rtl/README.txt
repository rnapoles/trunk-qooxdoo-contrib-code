===== qooxdoo-rtl =====

<note>
qooxodoo-rtl is developed against trunk and will be locked to different qooxdoo versions. 
Regularly updates will be commited.
</note>

qooxdoo-rtl is a fork of qooxdoo implementing rtl language support and it resides in github now.

Please follow the instructions below to manage setting up qooxdoo-rtl.

branch: master (trunk)

git clone -b master git@github.com:stefanand/qooxdoo-rtl.git
